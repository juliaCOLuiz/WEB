História da Web

A web é um serviço criado pelo Tim Bernes Lee,1990. Que funciona sobre a internet, que é uma rede de computadores, utilizado arquitetura Cliente- Servidor.

Para isso ele definiu a estrutura de funcionamento da web, cliente - servidor e criou a linguagem HTM, para exibição de documentos e o protocolo HTTP para comunicação entre aplicação cliente e servidor.

Para cuidar dos padrões da web r propriedade da tecnologia ele criou a fundação w3c World Web Consorcium

A referência das tags HTML pode ser encontrada no site da w3c HTML reference
