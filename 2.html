Estrutura e sintaxe do CSS

Uma regra CSS é um conjunto de instruções que define como um ou mais elementos HTML devem ser exibidos na página.

Ela é composta por três partes principais:

  • Seletor: indica qual elemento será estilizado (ex: button, .classe, #id)
  • Propriedades: características que você quer alterar (ex: color, background)
  • Valores: definem como a propriedade será aplicada (ex: red, 20px)
